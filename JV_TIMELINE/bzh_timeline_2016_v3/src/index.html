CHRONOTABLET // STERENNA

Interface de saut temporel ludique — timeline des sorties de jeux “majeurs” par mois. BZH CHRONICLES UI.

MODULE: G.A.M.E. / CHRONO-ARCHIVE

Principe : ta date réelle est projetée sur l’année cible (ex: 29/01/2026 → 29/01/2016). Les sorties à venir dans les 15 jours sont grisées. Au-delà : elles sont chiffrées et nécessitent de l’énergie pour être révélées.

SFX

1) Choisis un saut temporel

Une année cible. Pour le moment, 2016 est disponible.

2) Navigue par mois

Flèches + liste déroulante. Clique un jeu pour afficher sa fiche (Steam + image si dispo).

3) Débloque avec énergie

Les jeux masqués (> 15 jours) se débloquent au clic (1 énergie). Recharge: 5 / 24h.

4) Mémoire locale

Ton mois + tes reveals + ton énergie restent sur ce navigateur (localStorage).

Sélection de l’année
Les ingénieurs de Sterenna travaillent assidûment pour rendre disponibles d’autres sauts temporels.
Tip : au besoin, tu peux rendre cette page “kiosk” (plein écran) en ajoutant ce fichier dans une WebView/overlay.
