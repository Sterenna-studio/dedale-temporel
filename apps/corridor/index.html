Section S.T.E.A.M. – Accès restreint
Faites glisser le couloir à la souris ou avec le doigt pour explorer toutes les portes. Sur mobile, les portes s'alignent dans la profondeur du couloir.

Bienvenue dans le Dédale

Vous venez de pénétrer dans l’un des couloirs perdus de l’agence S.T.E.A.M. Ici, chaque porte est une anomalie temporelle ou l’accès à une salle de l’agence : certaines sont encore utilisées par les agents, d’autres ont été condamnées, parfois pour de bonnes raisons. Faites coulisser le couloir pour toutes les voir. En choisissant une porte, vous consultez son dossier d’énigme : son histoire, son usage et le code qui permet de l’activer (si elle n’est pas verrouillée). Ne vous perdez pas dans ces dédales : certaines portes mènent à des outils essentiels, d’autres à des expériences oubliées… et l’une d’elles vous ramènera vers la sortie de l’agence.
Console d’accès
