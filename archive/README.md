Rien ici n'est servi sur `nitro.sterenna.fr/dedale` (le workflow ne copie que les
sources actives). Aucun fichier n'a été perdu — déplacé ici plutôt que supprimé.

## Contenu

- **`impirmerie_medical/med_imprimerie_puzzle_v7` … `v7_8`** — itérations successives

  (restée dans `../impirmerie_medical/`). Ces dossiers sont l'historique de développement
  (ils peuvent différer entre eux — ce ne sont pas des copies identiques).
  - `med_imprimerie_puzzle_v7_6_2_workin` : version de travail intermédiaire.
- **`dedale_sfx_pack_cuivre_vapeur/`** — variante du pack sonore (contient un

## Restaurer un élément

Pour réactiver une version, la déplacer hors de `archive/` et brancher sa source
dans `.github/workflows/deploy-nitro.yml`.
