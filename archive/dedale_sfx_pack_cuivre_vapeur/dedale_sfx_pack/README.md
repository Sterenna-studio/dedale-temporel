# Dédale – Sound Pack (cuivre / vapeur / chaleureux)

## Fichiers (drop-in)
- **dedale_bg.mp3** : ambiance continue (hum + vapeur + cliquetis léger)
- **console_beep.wav** : sélection/validation (UI)
- **success.wav** : validation OK (tonal)
- **error.wav** : refus / mauvaise action (tonal)
- **door_hover.wav** : micro mécanique au survol
- **door_open.wav** : clac + souffle vapeur (mix)
- **door_locked.wav** : verrou lourd / tentative refusée
- **note_rustle.wav** : froissement / ouverture de note

## Mapping rapide (JS)
Voir `mapping.json`.

## Format
- BG: MP3 44.1kHz stéréo (45s, loopable)
- SFX: WAV 44.1kHz stéréo, 16-bit
